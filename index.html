Full Stack Apps

Bill Pay

In today's world a lot of businesses run off of subscriptions, monthly charges for their goods or services. For some people they can have so many they might forget how much they are spending on these subscriptions. So we created BillPay, an app thats allows users to keep track of their monthly bills so they can see how much they have to spend on other things.

Bill Pay allows users to add all of their monthly bills and adds them together for a grand total. The data the user puts in is saved so the user can come back to it at any time, we also added a income calculator so users can figure out their weekly, monthly, and yearly income.

Speakeasy

Speakeasy drinks is a informative app with over 700 differnet alcoholic and non-alcoholic drinks, but mostly alcoholic. Users are able to search for drinks two different ways, by ingredient or by name. If the user searches by name, they can pull up every drink with the same letters, word, or even words. If the user searches by ingredient, it will pull up all the drinks with that ingredients whether it's a type of alcohol or an addative ingredient like olives or lemon. Users are also able to use the random search, it will pull up one random drink for the user. For any of these options the user chooses it the results pull up each drinks name and image and if the user click on it it will pull up the ingredients, measuremnets, and instructions for making the drink. We also added a login and favorite function. Users are able to create logins and favorite drinks. then can then use the search favorite function to pull up drinks they have favorited at a later time.
